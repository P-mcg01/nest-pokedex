# Ejecutar en desarrollo

1. Clonar repositorio
2. Ejecutar

```properties
npm install
```

3. Tener Nest CLI instalado

```properties
npm i -g @nestjs/cli
```

4. Levantar la base de datos

```properties
docker compose up -d
```

## Stack usado

- MongoDB
- Nest
